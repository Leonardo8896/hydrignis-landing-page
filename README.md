# Objetivo
O objetivo do Hydrignis é desenvolver um sistema integrado capaz de monitorar incêndios e transformar água imprópria em potável, utilizando sensores inteligentes, automação via IoT e energia solar. A proposta é oferecer soluções acessíveis, sustentáveis e eficientes para comunidades, empresas e instituições, promovendo segurança, economia de recursos e impacto social real. Por meio de dispositivos autônomos e um aplicativo centralizado, o projeto busca facilitar o controle, reduzir riscos ambientais e garantir o acesso à água de qualidade em diferentes cenários.
